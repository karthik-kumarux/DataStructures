Understanding CSS

This page demonstrates different ways to style HTML elements using CSS (Cascading Style Sheets). We'll cover inline, internal, and external CSS.

Inline CSS

Inline CSS is applied directly to an HTML element using the style attribute. It's useful for applying unique styles to a specific element, but it's not recommended for styling large sections of a website because it can make the HTML code harder to read and maintain.

Internal CSS

Internal CSS

Internal CSS is defined within a <style> tag in the <head> section of an HTML document. It's used to define styles for a single HTML page. It's more organized than inline CSS, but it's not ideal for styling multiple pages.

External CSS

External CSS is defined in a separate file (with a .css extension) and linked to the HTML document using the <link> tag. This is the most common and recommended way to style HTML pages. It allows you to control the appearance of multiple pages from a single CSS file, making your website easier to maintain and update.
